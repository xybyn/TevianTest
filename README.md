# TevianTest

Build on linux:
```
mkdir build-linux
cd build-linux
cmake ..
cmake --build .
./TevianTest
```

Build on Windows:
```
mkdir build
cd build
cmake ..
cmake --build .
Debug/TevianTest.exe
```
